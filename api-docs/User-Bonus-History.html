User Bonus History

URI: /json/userBonusHistory.php


Shows a history of bonus points and wedges




Parameter Data type Additional information
other_userid int The userid of the other side of the transaction for gifts (points or wedges)
type List Which element(s) to show:

giftPoints
giftWedge
wedgePF
wedgeGFL
torrentThanks
millionaires


Example input

type[]=giftWedge&type[]=wedgePF&type[]=wedgeGFL


Output Parameters
