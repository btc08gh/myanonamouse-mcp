Load User Data

URI: /jsonLoad.php


This API point is for loading user data on the fly


Quickly gets updated user data on the fly without the workload of a full page load.

Parameter Data type Additional information
clientStats empty If set (and id is not set), adds in list of clients, how many torrents, and connectable status (30 min cache time)
id int userid for whom to get the data
notif empty If set (and id is not set), adds in the notifications (top banner area normally) to the returned information
pretty empty If set (and id is not set), outputs with JSON_PRETTY_PRINT to make it more "legible" for working with it.
snatch_summary empty If set (and id is not set), adds the basic torrent breakdown from the snatch summary.
