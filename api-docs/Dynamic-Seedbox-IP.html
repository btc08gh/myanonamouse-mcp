Dynamic Seedbox IP

URI: https://t.myanonamouse.net/json/dynamicSeedbox.php


Set your dynamic seedbox ip


Rate limit: once per hour (rolling window)


Compendium of information, scripts, and docker using this API



We recommend using MouseHole via docker as the simplest method


This endpoint can only be used with an IP or ASN locked session (mam_id) with the allowance to set the dynamic seedbox ip. This sessions must be setup in the security portion of user preferences, as it is different than the one your browser uses.



Example input

No input data required.


Output Parameters

Parameter Data type Additional information
AS string Organization name associated with the IP address
asn int ASN (see https://en.wikipedia.org/wiki/Autonomous_system_(Internet) ) associated with the IP address
ip string IP address the call was made from
msg string Meaning of responses:
Http Code Message Meaning
200 No Change The ip you accessed from is already the one set
200 Completed The system has finished the update
429 Last Change too recent The update has been quashed because you've changed too recently.
403 No Session Cookie You didn't properly provide the mam_id session cookie
403 Invalid session The system deemed the session invalid (bad mam_id value, or you've moved off the locked IP/ASN
403 Invalid session - IP mismatch The session is locked to a single IP address, and you are not accessing from it.
403 Invalid session - ASN mismatch The session is locked to a list of one or more ASN, but you are accessing from a different one
403 Invalid session - Invalid Cookie The system could not decode the cookie. Bad/corrupted value
403 Incorrect session type - not allowed this function The session cookie is for a session that is not allowed to set dynamic seedbox.
403 Incorrect session type - non-API session The session cookie is for a full normal web session, which is not allowed to set the dynamic seedbox
Success boolean If the operation was a success or failure.

Additional Information

This is designed for automated use, such as on VPN tunnel up in open vpn.

One could initialize this with something such as:

curl -c /path/docker/persists/mam.cookies -b 'mam_id=long________session_______string' https://t.myanonamouse.net/json/dynamicSeedbox.php


Then called in the future (in say a script file):

/path/docker/persists/tun_up.sh

#!/bin/bash
curl -c /path/docker/persists/mam.cookies -b /path/docker/persists/mam.cookies https://t.myanonamouse.net/json/dynamicSeedbox.php
